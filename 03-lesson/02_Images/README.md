# Images

We can use images from the internet and from the local computer. 

### img
src (source of the image, the URL or local name)
מקור התמונה
```html
<img src=""/>
```
#### Attributes of image:
1. height - will define the height of an image
גובה התמונה

2. width - will define the width of an image
רוחב התמונה
